Développement web - HTML

Introduction

Bienvenue dans le monde du développement web !

Si vous êtes nouveau dans cet univers, c'est un excellent point de départ pour comprendre comment tout fonctionne derrière les pages web que vous visitez chaque jour. Imaginez HTML comme la fondation d'une maison : c'est la structure sur laquelle tout est construit. Sans HTML, il n'y aurait pas de site web.

HTML, c'est quoi exactement ?

HTML, ou HyperText Markup Language, est un langage qui permet de décrire la structure d'une page web. Contrairement à d'autres langages de programmation, il ne s'agit pas ici de dire à l'ordinateur quoi faire, mais plutôt de lui expliquer comment afficher et organiser les éléments d'une page : les titres, les paragraphes, les images, les liens, etc. Bref, tout ce que vous voyez lorsque vous ouvrez un site web.

Exemple 1:

<!DOCTYPE html> <html lang="fr"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Afficher du code HTML</title> </head> <body> <!-- Ceci est un commentaire --> <p>Voici un exemple de code HTML :</p> </body> </html>
